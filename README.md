## Overview
This project is a responsive personal portfolio website developed using HTML, CSS, and JavaScript.  
It represents my profile as a Software Engineering student and showcases selected academic projects.

## Features

- Dark mode toggle
- Projects section with icons
- Skills section (Technical & Soft Skills)

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any modern browser.
3. Use the navigation bar to explore sections.
4. Toggle dark mode using the moon icon.
